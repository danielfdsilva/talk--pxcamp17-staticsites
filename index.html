The tools

  • Metalsmith - our generator
  • Github and Github Pages - for hosting
  • Prose - the content manager
  • Circle CI - continuous integration service

Metalsmith

  • An extremely simple, pluggable static site generator.
  • Makes no broad assumptions. All logic is handled by plugins.
  • Since everything is a plugin, the core library is actually just an abstraction for manipulating a directory of files.

Github and Github Pages

  • Free version control allowing collaboration
  • Good hooks and integrations with other services
  • Free static hosting for OS projects

Prose.io

  • Simple online file editor, specialized in markdown files
  • Powered by github

Circle CI

  • Continuous integration service
  • Free for OS projects
